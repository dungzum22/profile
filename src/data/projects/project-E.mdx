# Introduction

Platform Flower Shop is an e-commerce website designed specifically for the flower business. The platform enables multiple users to register as sellers or buyers, making it easy to trade various types of flowers in a user-friendly and efficient environment.
## Key Features

- **User Registration & Account Management**: Both buyers and sellers can create and manage their accounts.
- **Personal Storefronts for Sellers**: Each seller has a personalized store to showcase their flower products.
- **Product Management**: Sellers can add, edit, and remove flower listings with descriptions, images, and pricing.
- **Product Search & Filtering**: Buyers can search and filter flowers by type, price, name, or seller.
- **Shopping Cart & Order Placement**: Includes core e-commerce functionalities like adding to cart and placing orders.
- **Ratings & Reviews**: Buyers can leave feedback and rate the products and sellers.

## Technical Insights

- **Frontend**: Built with React and Next.js for a responsive, modern user interface.
- **Backend**: Developed using C# and RESTful APIs to handle business logic and data operations.
- **Database**: SQL Server used to store user, product, and order data securely.
- **Architecture**: Modular design allowing future scalability and integration with third-party services like payment gateways.

## Challenges and Solutions

- **Multi-role System**: Implemented clear role-based access control for buyers and sellers.
- **Real-time Updates**: Designed efficient database queries and API calls to keep store data up to date.
- **User Experience**: Focused on UI/UX best practices to make the shopping and selling process intuitive.

## Outcome

The Flower Shop Platform provides a complete marketplace experience for florists and flower enthusiasts. It supports multiple users, streamlines the buying/selling process, and serves as a solid foundation for a scalable flower trading ecosystem.
